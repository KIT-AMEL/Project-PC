Overview

This system loads, cleans, sorts, and visualizes point clouds from CSV files. It removes duplicates, sorts using a Nearest Neighbor Search approach, and allows users to toggle between scatter or line plots. Results can be exported as CSV. Nearest Neighbor Search, often used in computational geometry and machine learning, learns spatial relationships dynamically and sorts accordingly.

Required CSV Format

        X,Y,Z
        1.0,2.0,10.0
        3.2,4.5,5.0
        5.1,1.8,7.0
        ...

If the Z column is missing, it is auto-filled with 0.
